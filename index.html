About Me

Ik ben een IT Professional. Ik begrijp dat een IT professional de schakel is tussen de gebruikers en de informatievoorziening. Momenteel werk ik als Functioneel beheerder binnen de gemeente Utrecht.

Als een oude ICTer heb ik altijd passie voor de nieuwste ontwikkelingen in IT gebied. Naast mijn werk heb ik in mijn vrije tijd een opleidingstraject Data-analist gedaan, waarin ik mij heb gespecialiseerd in data-analyse met tools zoals Power BI, SQL en Python. Daarnaast heb ik ruim ervaring van web ontwikkeling en ben ik goed bekend met Javascript, HTML en CSS. Mijn achtergrond in informatica en passie voor continu leren zorgen ervoor dat ik nieuwe uitdagingen met enthousiasme aanpak. Van nature ben ik nauwkeurig, analytisch sterk en help ik anderen graag. Ik ben ervan overtuigd dat ik een waardevolle toevoeging kan zijn voor elke organisatie.
